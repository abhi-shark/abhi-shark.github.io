Fertilizer Statistical Analysis Project

3 levels of Fertilizers and Watering Frequencies were investigated relative to plant height in 30 days, using a Randomized Complete Block Design with differing plant blocks. Analysis was done in SAS using ANOVA with blocking & interaction effects.

Lubrication Simulation Project

Simulated the fluid film pressure distribution for a Pivoted-Pad Thrust Bearing using MATLAB and the Newton-Raphson method, and compared the results to theoretical and lab results.

ICE Engine Simulation using GT-POWER

Simulated naturally aspirated, turbocharged, supercharged ICEs (gasoline, diesel), along with BEVs, PHEVs and FCEVs using GT-POWER, and compared the results to lab runs using a 3.3L Duratec V6 gasoline and a 3.0L Powerstroke V6 diesel engine.
